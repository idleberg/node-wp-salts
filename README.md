Generates an object of default WordPress salts (or any other string)

## Installation

`yarn add wp-salts || npm install wp-salts -S`

## Usage

Use ES6 imports or `require()` to include the module:

```js
const { wpSalts } = require('makensis');

wpSalts();
```
